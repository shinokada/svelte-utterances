# svelte-utterances

## Demo

[Svelte-Utterances Demo](https://svelte-utterances.vercel.app/)

## Installation

```sh
npm i -D @codewithshin/svelte-utterances
```

## Props

| Name        | Default        |
| ----------- | -------------- |
| `reponame`  | None           |
| `issueTerm` | `pathname`     |
| `label`     | `comments`     |
| `theme`     | `github-light` |

### Themes

The default is github-light. Select one from:

- github-light
- github-dark
- preferred-color-scheme
- github-dark-orange
- icy-dark
- dark-blue
- photon-dark
- boxy-light
- gruvbox-dark

### Issue term

The default is pathname. Select one from:

- url
- title
- og:title
- [Enter issue number]
- [Enter term]
